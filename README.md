## Function list

## Install

e.g: use `@uni-use/speak`

```shell
# by pnpm
pnpm install @uni-use/speak

# by npm
npm install -D @uni-use/speak

# by yarn
yarn add @uni-use/speak
```

## Usage

```js
import { speak } from '@uni-use/speak'

speak('Hello World!')
```
