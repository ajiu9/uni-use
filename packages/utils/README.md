## Install

```bash
# use pnpm
$ pnpm install -D @uni-use/utils

# use yarn
$ yarn add -D @uni-use/utils

# use npm
$ npm install -D @uni-use/utils
```

## Usage

```js
import { functionName } from '@uni-use/utils'

// run functionName
functionName()
```
